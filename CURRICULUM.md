## Course At A Glance

| Category | Details |
|---|---|
| Total Duration | 40 Hours (10 modules × ~4 hours each) |
| Delivery Format | Instructor-led + Self-paced hybrid |
| Target Audience | Beginners with no prior ML or AI experience |

---

## Module 1 — Introduction to AI & Machine Learning (3 hrs)

### Topics
- What is Artificial Intelligence? Definitions and scope
- Brief history: from Alan Turing to modern AI
- Subfields: Machine Learning, Deep Learning, NLP, Computer Vision
- How ML works: data → algorithm → model → prediction
- Types of ML: supervised, unsupervised, reinforcement learning
- Real-world applications across industries

### Activities
- Lecture: AI in everyday life (video + discussion)
- Group exercise: identify ML use cases in daily scenarios
- Quiz: Module 1 key concepts

### Resources
- Slides, introductory videos, curated reading list

---

## Module 2 — Python for Machine Learning (4 hrs)

### Topics
- Python basics: variables, data types, loops, functions
- NumPy: arrays, vectorized operations
- Pandas: DataFrames, loading and inspecting data
- Matplotlib & Seaborn: plotting and visualization basics
- Jupyter Notebooks: working environment setup

### Activities
- Hands-on lab: Python warm-up exercises
- Lab: Load and explore a real dataset with Pandas
- Lab: Create basic plots and charts

### Resources
- Python 3, Jupyter Notebook, Google Colab (cloud alternative)

---

## Module 3 — Data: Understanding & Preparation (4 hrs)

### Topics
- Types of data: structured, unstructured, categorical, numerical
- Exploratory Data Analysis (EDA): summary statistics, distributions
- Data quality: handling missing values, duplicates, outliers
- Feature engineering: encoding, scaling, normalization
- Train/test split and data leakage

### Activities
- Lab: EDA on a public dataset (Titanic, Iris)
- Lab: Clean and preprocess a messy dataset
- Assignment: Prepare a dataset for an upcoming model

---

## Module 4 — Supervised Learning: Regression (4 hrs)

### Topics
- What is regression? Predicting continuous values
- Simple linear regression: slope, intercept, line of best fit
- Multiple linear regression
- Polynomial regression
- Evaluation metrics: MAE, MSE, RMSE, R²

### Activities
- Lab: Build a house price prediction model
- Lab: Visualize regression results
- Quiz: Regression concepts

---

## Module 5 — Supervised Learning: Classification (4 hrs)

### Topics
- What is classification? Predicting discrete categories
- Logistic regression
- Decision trees and random forests
- K-Nearest Neighbors (KNN)
- Support Vector Machines (SVM) — conceptual overview
- Evaluation: accuracy, precision, recall, F1, confusion matrix

### Activities
- Lab: Email spam classifier
- Lab: Iris flower classification with multiple algorithms
- Assignment: Compare classifier performance on same dataset

---

## Module 6 — Unsupervised Learning (4 hrs)

### Topics
- What is unsupervised learning? No labeled data
- Clustering: K-Means algorithm step by step
- Choosing the right K: Elbow method
- Dimensionality reduction: PCA intuition and use cases
- Applications: customer segmentation, anomaly detection

### Activities
- Lab: Customer segmentation with K-Means
- Lab: Visualize high-dimensional data with PCA

---

## Module 7 — Model Evaluation & Improvement (4 hrs)

### Topics
- Bias-variance tradeoff
- Overfitting and underfitting
- Cross-validation: k-fold and stratified
- Hyperparameter tuning: Grid Search, Random Search
- Pipelines in scikit-learn
- Feature importance and selection

### Activities
- Lab: Diagnose and fix an overfitting model
- Lab: Improve a model using cross-validation and Grid Search

---

## Module 8 — Introduction to Neural Networks (5 hrs)

### Topics
- Biological inspiration: neurons and synapses
- The perceptron: weights, bias, activation functions
- Multi-layer perceptron (MLP) architecture
- Forward propagation and backpropagation (intuitive)
- Activation functions: ReLU, sigmoid, softmax
- Introduction to TensorFlow / Keras
- Overview of deep learning: CNNs, RNNs (conceptual)

### Activities
- Lab: Build a simple neural network with Keras
- Lab: Image classification on MNIST digit dataset
- Quiz: Neural network fundamentals

---

## Module 9 — Responsible AI & Ethics (3 hrs)

### Topics
- Bias in data and models: sources and consequences
- Fairness, accountability, and transparency (FAT)
- Privacy and data protection
- AI governance frameworks (EU AI Act overview, UNESCO guidelines)
- Explainable AI (XAI): why black-box models are a concern
- Case studies: ethical failures in AI deployments

### Activities
- Discussion: real-world AI bias case studies
- Group activity: audit a dataset for bias
- Reflection assignment: ethical considerations for a chosen AI use case

---

## Module 10 — Capstone Project & Deployment (5 hrs)

### Project Phases
1. **Problem Definition** — Select a real-world problem and dataset
2. **Data Preparation** — Clean, explore, and engineer features
3. **Modeling** — Train and compare at least two algorithms
4. **Evaluation** — Assess performance using appropriate metrics
5. **Deployment** — Serve the model via Flask API or Streamlit web app
6. **Presentation** — Present findings to the class

### Suggested Project Ideas
- Disease diagnosis prediction (medical dataset)
- Loan default prediction (financial dataset)
- Crop yield classification (agricultural dataset)
- Sentiment analysis on product reviews
